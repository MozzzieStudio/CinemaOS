# CinemaOS Custom Nodes for ComfyUI

Custom nodes that integrate CinemaOS with Fal.ai, Vertex AI, and the Vault.

## Installation

These are automatically copied to ComfyUI's custom_nodes folder during installation.

## Nodes

### Cloud Providers

- **FalProvider** - Generate images/videos via Fal.ai API
- **VertexProvider** - Generate images/videos via Google Vertex AI

### CinemaOS Integration

- **VaultContext** - Load character/location/prop context from the Vault
- **VaultSave** - Save generated outputs back to the Vault

### Utility

- **CreditTracker** - Track credit usage for cloud generation
